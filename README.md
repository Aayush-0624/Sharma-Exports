# Import Export Management Website

A full-stack web application designed for import-export businesses to manage product listings and customer inquiries efficiently.

## 🚀 Features

- User Authentication (Register/Login)
- JWT-based Authorization
- Product Management
- Customer Inquiry Management
- Admin Dashboard Functionality
- Responsive User Interface
- RESTful API Integration
- MySQL Database Support

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router
- Vite
- Tailwind CSS
- Framer Motion

### Backend
- Node.js
- Express.js
- JWT Authentication
- Bcrypt

### Database
- MySQL
